# Iconos para PWA

Para generar los iconos necesarios para la PWA, puedes usar una de estas herramientas online:

1. **PWA Icon Generator**: https://www.pwabuilder.com/imageGenerator
2. **RealFaviconGenerator**: https://realfavicongenerator.net/
3. **Favicon.io**: https://favicon.io/

## Iconos necesarios:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Instrucciones:

1. Crea un logo/icono para tu aplicación (512x512px recomendado)
2. Súbelo a alguna de las herramientas mencionadas arriba
3. Descarga todos los tamaños generados
4. Colócalos en esta carpeta `/public/icons/`

## Iconos temporales

Mientras tanto, se generarán iconos placeholder automáticamente en la primera build.
